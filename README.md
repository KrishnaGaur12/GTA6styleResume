# WANTED: Krishna Gaur - GTA VI Themed Resume 🚓🌴

An interactive, high-octane GTA VI themed landing page and developer portfolio built with React and GSAP. This project transforms a standard resume into an immersive visual experience worthy of Rockstar Games.

## 🌟 Features

- **Iconic Intro Sequence:** A meticulous recreation of the GTA VI style intro text, scaling and fading to reveal the main screen.
- **Dynamic Parallax Environments:** Interactive backgrounds that react to your mouse movement—the sky, the cityscape, and the character elements shift smoothly as you move your cursor.
- **"STILL RUNNING, NOT HUNTING" Section:** A dedicated hero section capturing the vibe of the franchise with bold typography and prominent download calls-to-action.
- **Quirky Resume Integration:** 
  - **WANTED Poster:** Contact info styled as a high-stakes bounty.
  - **Player Stats:** Technical skills categorised as Weapons, Arsenal, Stash, and Gadgets.
  - **Missions & Heists:** Professional experience and major projects (ByteCrush, AlgoVerse, ChargeShare) framed as significant operations.
  - **Trophies:** Awards and achievements displayed like unlocked gameplay milestones.

## 🛠️ Tech Stack

- **React:** Component-based UI architecture.
- **Vite:** High-performance local development server and bundler.
- **Tailwind CSS v4:** Rapid, utility-first styling for complex angled layouts and neon glow effects.
- **GSAP (GreenSock):** Industry-standard animation library powering the complex timelines, scroll triggers, and mouse-tracking parallax effects.

## 🚀 Getting Started

To run this heist locally on your machine:

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone <your-repo-url>
   cd GTA6
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   ```

4. **Access the site:**
   Open your browser and navigate to `http://localhost:5173` (or the port specified in your terminal).

## 🎨 Typography

This project relies on the classic GTA stylistic fonts (like `pricedown.otf`) to maintain authentic immersion. Make sure the font file is present in your `public` directory.
